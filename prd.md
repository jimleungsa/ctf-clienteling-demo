# 珠宝导购助手 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
珠宝导购助手是一款面向珠宝零售门店导购员的企业微信H5应用，旨在帮助导购高效管理客户、执行任务、查看经营数据，提升导购工作效率和销售转化率。

### 1.2 目标用户
- **主要用户**：珠宝门店导购员
- **使用场景**：门店日常销售服务、客户维护、任务执行、数据查看

### 1.3 核心价值
1. **任务驱动**：每日任务清晰呈现，导购明确知道要做什么
2. **客户洞察**：客户画像一目了然，精准服务
3. **数据赋能**：实时查看业绩数据，激励导购
4. **AI辅助**：智能话术生成，降低沟通门槛

## 2. 核心功能模块

### 2.1 五大导航模块

| 模块 | 核心功能 | 优先级 |
|------|---------|--------|
| 首页 | 每日任务、可触达人群包、门店会员数据指标 | P0 |
| 客户中心 | 我的客户列表、客户画像、客户详情 | P0 |
| 数据中心 | 任务/会员/企微/排名 四个数据维度 | P1 |
| 任务中心 | 群发任务、客户商机、AI话术生成 | P0 |
| 我的 | 个人业绩、基础信息、工作台快捷入口 | P1 |

### 2.2 任务类型定义

基于珠宝行业特性，任务类型包括但不限于：

| 任务类别 | 具体任务 | 说明 |
|---------|---------|------|
| **会员权益** | 会员生日关怀、会员等级到期提醒、积分到期提醒、专属权益通知 | 基于会员生命周期触发 |
| **营销活动** | 新品上市通知、促销活动邀约、VIP专属活动邀请、节日营销触达 | 基于营销日历触发 |
| **派券任务** | 优惠券发放、代金券推送、满减券提醒 | 基于营销策略触发 |
| **预约到店** | 预约提醒、到店确认、服务跟进 | 基于客户预约行为 |
| **复购激活** | 沉睡客户唤醒、高潜客户培育、流失预警挽回 | 基于RFM模型触发 |
| **企微运营** | 添加好友任务、朋友圈发布、社群互动 | 基于企微运营目标 |
| **服务任务** | 售后服务回访、保养提醒、以旧换新通知 | 基于服务周期触发 |

## 3. 页面详细设计

### 3.1 首页

**页面目标**：导购每天第一眼看到的页面，快速了解今日工作重点

**页面布局**：
- **顶部区域**：企业微信导航栏（返回工作台）+ 页面标题
- **数据概览卡**：今日待办任务数、可触达客户数、本月业绩完成率
- **今日任务区**：按优先级排序的任务卡片列表
- **人群包推荐**：基于算法推荐的今日可触达人群包
- **门店数据**：本店今日关键指标（客流、转化、客单价）

**交互逻辑**：
- 任务卡片点击 → 进入任务详情
- 人群包点击 → 进入客户列表（已筛选）
- 查看更多 → 进入任务中心/数据中心

---

### 3.2 客户中心

**页面目标**：导购管理自己客户的统一入口

**页面布局**：
- **搜索栏**：支持按姓名/手机号/标签搜索
- **筛选器**：按会员等级/消费金额/标签/最近消费时间筛选
- **客户列表**：
  - 客户微信头像
  - 客户姓名（微信昵称）
  - 会员等级（普通/银卡/金卡/钻石）
  - 财年消费金额（万元）
  - 消费单量
  - 1-3个智能标签（如：高价值、沉睡客户、新品偏好）
- **客户详情页**：
  - 基础信息（头像、姓名、会员等级、入会时间）
  - 消费数据（累计消费、年均消费、客单价、消费频次）
  - 偏好标签（品类偏好、价格带偏好、购买时段）
  - 互动记录（最近联系时间、最近购买时间、服务记录）
  - 快捷操作（拨打电话、发消息、创建任务）

**交互逻辑**：
- 列表项点击 → 客户详情页
- 搜索输入 → 实时筛选
- 筛选条件 → 列表刷新
- 快捷操作 → 唤起企微聊天/拨号

---

### 3.3 数据中心

**页面目标**：数据驱动决策，让导购了解自己的业绩和排名

**页面布局（5个子Tab）**：

#### Tab 1: 任务数据
- 当月任务完成数/总任务数
- 任务完成率趋势图（近30天）
- 任务类型分布（饼图）
- 任务完成排名（本店）

#### Tab 2: 会员数据
- 当月复购客户数/复购金额
- 当月首次消费客户数/金额
- 会员消费趋势（近12个月）
- 会员等级分布

#### Tab 3: 企微数据
- 当月新增企微好友数
- 累计企微客户数
- 当月流失客户数
- 客户互动频次统计

#### Tab 4: 排名数据
- 本店排名（销售额/任务完成率）
- 本区排名（同区域门店对比）
- 全国排名（TOP 100榜单）
- 排名趋势（上升/下降）

#### Tab 5: 业绩数据（可选）
- 个人销售额/目标完成率
- 个人客单价/连带率
- 个人转化率

**交互逻辑**：
- Tab切换 → 内容切换
- 时间筛选 → 数据刷新（日/周/月/年）
- 图表点击 → 下钻详情

---

### 3.4 任务中心

**页面目标**：导购执行任务的统一入口，区分群发任务和一对一商机

**页面布局**：

#### 顶部Tab切换：群发任务 | 客户商机

---

**群发任务板块**：
- 任务列表卡片：
  - 任务名称（如：母亲节活动邀约）
  - 任务类型标签（营销活动）
  - 截止时间（倒计时）
  - 完成进度（已完成/总数）
  - 任务状态（待执行/进行中/已完成）
- 任务详情页：
  - 任务说明
  - 推荐话术
  - 素材图片/链接
  - 目标客户列表（分页加载）
  - 一键群发按钮

---

**客户商机板块**：
- 商机列表卡片：
  - 商机主题（如：积分到期提醒）
  - 人群包数量
  - 触达占比（已联系/总数）
  - 优先级标签（高/中/低）
- 商机详情页：
  - 商机说明（为什么这批客户需要联系）
  - 预计触达客户列表（分页加载）
  - 每个客户项点击 → AI话术生成页

---

**AI话术生成页**：
- 页面标题：AI话术助手
- 主题说明（如：为高价值客户生成积分到期提醒话术）
- 客户信息摘要（姓名、积分余额、到期时间）
- AI生成区域：
  - 生成按钮
  - 3条话术选项（不同风格：正式/亲切/促销）
  - 每条话术可复制
  - 重新生成按钮
- 快捷操作：复制话术 → 跳转企微聊天

**交互逻辑**：
- 任务/商机卡片点击 → 详情页
- 客户列表滚动 → 分页加载
- AI生成 → 展示3条话术
- 复制话术 → Toast提示 + 唤起企微

---

### 3.5 我的

**页面目标**：个人中心，查看个人信息和工作台快捷入口

**页面布局**：

#### 个人信息区
- 企微头像
- 导购姓名
- 工号
- 所在分店信息（分店行号 + 分店名称）

#### 业绩数据区
- 本月销售额
- 本月目标完成率
- 本月任务完成数
- 点击查看详情 → 数据中心

#### 工作台快捷入口
- **金价查询**：实时金价/银价查看
- **素材库**：产品图片/视频/文案素材
- **话术库**：常用话术/场景话术/异议处理
- **O2O选配**：线上下单门店提货/配送
- **培训学习**：产品知识/销售技巧/考试
- **客服支持**：常见问题/在线客服

#### 设置区
- 消息通知设置
- 隐私设置
- 关于我们
- 退出登录

**交互逻辑**：
- 工作台入口点击 → 对应功能页面/H5
- 业绩数据点击 → 数据中心
- 设置项点击 → 对应设置页面

## 4. 用户交互流程

### 4.1 典型使用场景

#### 场景1：导购早晨到岗
```
企微工作台 → 珠宝导购助手 → 首页
查看今日任务 → 点击任务 → 查看话术/素材
→ 一键群发/逐个触达 → 完成任务
```

#### 场景2：客户维护
```
客户中心 → 搜索客户/筛选客户 → 客户详情
→ 查看客户画像 → 点击发消息
→ 选择商机主题 → AI生成话术 → 复制发送
```

#### 场景3：查看业绩
```
数据中心 → 切换Tab查看不同维度
→ 查看排名 → 查看趋势
→ 我的 → 查看个人业绩
```

#### 场景4：使用工具
```
我的 → 工作台 → 金价查询/素材库/话术库
→ 使用对应功能
```

## 5. 非功能性需求

### 5.1 性能要求
- 页面首屏加载时间 < 2秒
- 列表分页加载，每页20条
- 图片懒加载

### 5.2 兼容性要求
- 支持企业微信内置浏览器
- 支持主流移动端浏览器（iOS Safari、Android Chrome）
- 适配 iPhone/Android 主流机型

### 5.3 安全要求
- 企微身份认证
- 数据传输加密
- 敏感数据脱敏显示

### 5.4 体验要求
- 操作反馈明确（Toast提示）
- 加载状态可视化
- 空状态友好提示
- 错误状态可恢复

## 6. 数据需求

### 6.1 需要展示的数据字段

| 数据类型 | 字段 | 来源 |
|---------|------|------|
| 客户数据 | 微信头像、昵称、姓名、手机号、会员等级、累计消费、消费单量、标签 | CRM系统 |
| 任务数据 | 任务名称、类型、截止时间、目标客户数、完成数、话术、素材 | 任务系统 |
| 业绩数据 | 销售额、目标、完成率、排名、客单价、转化率 | BI系统 |
| 企微数据 | 好友数、新增数、流失数、互动频次 | 企微API |
| 金价数据 | 黄金/铂金/钻石实时价格 | 金价接口 |
