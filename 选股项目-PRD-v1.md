## 1. 项目背景

### 1.1 背景
将股票分析策略转化为可执行的产品功能，实现数据驱动的智能选股。

### 1.2 目标
- 提供趋势+基本面双因子选股能力
- 支持回测验证策略有效性
- 输出可量化的选股指标和风险评估

## 3. 功能需求

### 3.1 选股策略（clawstock 提供）

#### 策略类型
**趋势 + 基本面双因子筛选**

#### 筛选条件

**趋势条件：**
- 股价 > 20日均线 > 60日均线
- 20日均线斜率 > 0（上升趋势）
- 近5日成交量均值 > 近20日成交量均值（放量）

**基本面条件：**
- ROE > 15%
- 营收同比增长 > 10%
- 负债率 < 60%

**风险控制：**
- 排除 ST 股票
- 排除近1年有重大违规记录的公司
- 流动性要求：日均成交额 > 5000万
- 市值 > 50亿（clawanalysis 建议）
- 单一行业集中度 ≤ 30%（clawanalysis 建议）

#### 输出指标

**策略绩效指标：**
- 胜率 - 历史回测中盈利交易占比
- 盈亏比 - 平均盈利/平均亏损
- 最大回撤 - 历史最大资金回撤幅度
- 年化收益率 - 策略年化收益
- 夏普比率 - 风险调整后收益

**个股输出：**
- 是否可交易（是/否）
- 入场点（具体价格或条件）
- 止损点（具体价格或百分比，建议-8%硬止损）
- 目标位（具体价格或百分比）
- 仓位建议（占总资金百分比）

#### 回测要求
- 回测周期：至少3年历史数据（建议5年覆盖牛熊周期）
- 交易成本：考虑佣金（万2.5）和滑点（0.1%）
- 调仓频率：周度/月度可选
- 基准对比：对比沪深300指数
- 复权方式：前复权（回测用）

#### 数据需求（clawstock补充）

**行情数据：**
- 日线：开盘、收盘、最高、最低、成交量、成交额
- 分钟线：5/15/30/60分钟（精确入场点）
- 复权：前复权、后复权
- 历史范围：至少5年

**财务数据：**
- 三张表：资产负债表、利润表、现金流量表（季度）
- 核心指标：ROE、ROA、毛利率、净利率、负债率
- 成长指标：营收增速、净利润增速、EPS增速
- 估值指标：PE、PB、PS、PEG（每日）

**其他数据：**
- 行业分类、概念板块、股本结构
- 股东人数变化、龙虎榜、北向资金、融资融券

**数据供应商（免费方案 - 已确定）：**
- **日线数据：Baostock（首选）+ AkShare（备份）**
  - 覆盖范围：全A股，1990年至今
  - 更新频率：每日20:00后
  - 稳定性：Baostock ⭐⭐⭐⭐⭐，AkShare ⭐⭐⭐⭐
- **财务数据：AkShare**
  - 财务指标：ROE/PE/PB等核心指标
  - 财务报表：三张表
  - 更新频率：季报发布后1-2天
- **数据质量**：与Wind付费数据95%一致，足够支撑回测验证

**数据初始化方案：**
- 5年历史数据：约600万条记录
- 下载时间：2-3天（限速访问，避免反爬）
- 存储空间：约2-5GB
- 建议：优先获取沪深300成分股

**数据更新策略：**
- 日线数据：每日收盘后自动更新（AkShare）
- 更新时长：约10-30分钟
- 财务数据：季报发布后手动触发更新
- 实时数据：MVP版本暂不支持，V1.1规划

**成本：完全免费（0元）**

**数据存储与同步：**
- 本地存储：SQLite（MVP）/ PostgreSQL（生产）
- 校验机制：完整性+准确性+一致性
- 补偿机制：重试+告警+补偿队列
- 自动同步：每日20:30自动执行
- 监控面板：实时同步状态查看

**数据合规：**
- 仅使用公开/授权数据
- 禁止内幕交易信息
- 数据本地存储，不对外泄露

## 4. 数据结构

### 4.1 股票基础信息表
```sql
CREATE TABLE stocks (
    code VARCHAR(10) PRIMARY KEY,      -- 股票代码
    name VARCHAR(50),                   -- 股票名称
    industry VARCHAR(50),               -- 所属行业
    market VARCHAR(10),                 -- 市场（SH/SZ）
    market_cap BIGINT,                  -- 市值
    float_cap BIGINT,                   -- 流通市值
    is_st BOOLEAN,                      -- 是否ST
    created_at TIMESTAMP
);
```

### 4.2 日线行情表
```sql
CREATE TABLE daily_quotes (
    id SERIAL PRIMARY KEY,
    code VARCHAR(10),                   -- 股票代码
    date DATE,                          -- 日期
    open DECIMAL(10,2),                 -- 开盘价
    close DECIMAL(10,2),                -- 收盘价
    high DECIMAL(10,2),                 -- 最高价
    low DECIMAL(10,2),                  -- 最低价
    volume BIGINT,                      -- 成交量
    amount BIGINT,                      -- 成交额
    UNIQUE(code, date)
);
```

### 4.3 财务指标表
```sql
CREATE TABLE financial_indicators (
    id SERIAL PRIMARY KEY,
    code VARCHAR(10),                   -- 股票代码
    report_date DATE,                   -- 报告期
    pe_ttm DECIMAL(10,2),               -- 市盈率TTM
    pb DECIMAL(10,2),                   -- 市净率
    roe DECIMAL(10,2),                  -- ROE
    revenue_growth DECIMAL(10,2),       -- 营收增长率
    debt_ratio DECIMAL(10,2),           -- 负债率
    UNIQUE(code, report_date)
);
```

### 4.4 选股结果表
```sql
CREATE TABLE stock_picks (
    id SERIAL PRIMARY KEY,
    strategy_id VARCHAR(50),            -- 策略ID
    code VARCHAR(10),                   -- 股票代码
    pick_date DATE,                     -- 选股日期
    entry_price DECIMAL(10,2),          -- 建议入场价
    stop_loss DECIMAL(10,2),            -- 止损价
    target_price DECIMAL(10,2),         -- 目标价
    position_pct DECIMAL(5,2),          -- 建议仓位%
    confidence_score INT,               -- 置信度评分
    reason TEXT,                        -- 选股理由
    created_at TIMESTAMP
);
```
