# 选股项目 - 免费数据来源方案

**需求**: 零成本获取A股数据  
**日期**: 2026-03-22

---

## 1. 免费数据方案汇总

### 方案一：AkShare（推荐）

**优势：**
- 完全免费，开源项目
- A股数据覆盖全（日线、分钟线、财务数据）
- Python接口，易于集成
- 社区活跃，持续更新

**数据范围：**
- 日线行情：全部A股历史数据
- 分钟线：5/15/30/60分钟
- 财务数据：三张表+核心指标
- 实时数据：延迟15分钟

**限制：**
- 稳定性不如商业API
- 需要处理反爬（频率限制）
- 数据更新延迟（日线T+1）

**技术方案：**
```python
import akshare as ak
# 日线数据
stock_zh_a_hist_df = ak.stock_zh_a_hist(symbol="000001", period="daily")
# 财务数据
stock_financial_report_sina_df = ak.stock_financial_report_sina(stock="600000")
```

**数据范围：**

- 实时行情：延迟15秒
- 财务数据：基础指标

**限制：**

## 2. 推荐方案

### MVP版本（免费）

## 3. 数据获取计划

### 初始化（一次性）
